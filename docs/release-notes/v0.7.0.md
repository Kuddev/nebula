# Nebula Terminal 0.7.0

## English

Nebula Terminal 0.7.0 adds wallpaper support, a live appearance preview, dropdown option lists, and font-size controls, and fixes a long list of appearance, input, and math-rendering issues.

### Added

- Added background image support: choose a wallpaper for the terminal with stretch mode, position, and image opacity, optionally covering the whole window.
- Added a startup-directory setting so new terminal tabs open in a directory of your choice.
- Added window size memory: Nebula reopens with the same size and maximized state as last time.
- Added a live preview card to the Appearance page that immediately reflects color, font, font size, and cursor changes.
- Added dropdown option lists for every multi-choice setting, replacing click-to-cycle rows.
- Added font size controls: a numeric control in settings and Ctrl + mouse wheel zoom in the terminal, with the value remembered.
- Added a background color picker with preset colors and hex input.
- Added selectable cursor shapes with blinking on by default.
- Added copy-on-select, on by default, with right-click copy or paste.
- Added SSH keepalive so idle remote sessions no longer get disconnected.

### Fixed

- Fixed wallpaper fading toward the desktop behind the window; lowering image opacity now fades into the theme background — lighter in light mode, darker in dark mode.
- Fixed PNG, JPG, WebP, and BMP wallpapers not all displaying, wallpapers covering the terminal's rounded corners, and the preview card not showing the real wallpaper.
- Fixed uneven window transparency, the missing restore icon when maximized, and background text showing through the command palette.
- Fixed formulas typed into AI tools being rendered while still editable, rendered formulas reverting to raw text while scrolling, and leftover text such as `&nbsp;` inside matrices.
- Fixed the terminal keeping colors from the previous theme after switching between light and dark.
- Fixed tab titles not following directory changes while renamed tabs keep their names.
- Fixed custom fonts producing hollow boxes and broken symbols.
- Fixed SSH warnings that could not be closed and spilled over the sidebar.
- Fixed the title-bar buttons to form a seamless Windows-style group with the familiar red close hover, clickable in the very corner when maximized.
- Fixed the working spinner pausing between rotations, sliders showing a resize cursor, icons sitting off-center, and command palette rows not highlighting under the mouse.
- Fixed the Ctrl+click link tooltip jumping around with the pointer.
- Fixed cursor shape and blinking changes requiring a restart.
- Fixed confirmation dialogs stretching into wide banners; messages now wrap over a compact Yes / No dialog.
- Fixed harmless clipboard warnings when another program briefly held the clipboard.

### Improved

- Improved long sessions with many formulas: scrolling stays fast and memory use stays stable.
- Improved scrolling smoothness with custom fonts.
- Improved battery and CPU usage: animations only redraw while something is moving.
- Improved consistency of settings controls, which now share one implementation everywhere.

## 中文

Nebula Terminal 0.7.0 新增壁纸、外观实时预览、下拉选项列表和字号调节，并集中修复了大量外观、输入和公式显示问题。

### 新增

- 新增背景图支持：可以为终端设置壁纸，调整拉伸方式、位置和图片不透明度，也可以铺满整个窗口。
- 新增启动目录设置，新建终端标签页可以在指定目录中打开。
- 新增窗口大小记忆，Nebula 会以上次的大小和最大化状态重新打开。
- 新增外观页实时预览卡片，颜色、字体、字号和光标改动立刻可见。
- 新增所有多选项设置的下拉选项列表，不再点击循环切换。
- 新增字号调节：设置中的数字控件和终端里的 Ctrl+滚轮缩放，数值会被记住。
- 新增背景色选择器，提供预设颜色和十六进制输入。
- 新增可选的光标形状，光标闪烁默认开启。
- 新增选中即复制（默认开启），右键可复制或粘贴。
- 新增 SSH 保活，空闲的远程会话不再被断开。

### 修复

- 修复降低壁纸不透明度时透出窗口后面桌面的问题；现在会淡入主题背景色——浅色模式越来越浅、深色模式越来越深。
- 修复部分 PNG、JPG、WebP、BMP 壁纸无法显示、壁纸盖住终端圆角、预览卡不显示真实壁纸的问题。
- 修复窗口透明度不均、最大化后还原图标缺失、命令面板透出底层文字的问题。
- 修复在 AI 工具中输入公式时被提前渲染、滚动时公式变回原始文字、矩阵中残留 `&nbsp;` 文字的问题。
- 修复切换明暗主题后终端残留旧主题颜色的问题。
- 修复标签页标题不跟随目录变化的问题；重命名过的标签页保持自定义名称。
- 修复自定义字体出现空心方框和乱码的问题。
- 修复 SSH 警告无法关闭、背景延伸到侧边栏的问题。
- 修复标题栏按钮，现在是无缝的 Windows 风格按钮组，关闭按钮悬停显示红色，最大化时角落也能点到。
- 修复转圈动画停顿、滑块显示双向箭头光标、图标偏离中心、命令面板候选行不高亮的问题。
- 修复 Ctrl+点击 链接提示跟随指针乱跳的问题。
- 修复光标形状和闪烁设置需要重启才生效的问题。
- 修复确认对话框被拉成横幅的问题，长文字会自动换行，按钮为简单的"是 / 否"。
- 修复其他程序短暂占用剪贴板时弹出无意义警告的问题。

### 改进

- 改进含大量公式的长会话，滚动保持流畅、内存稳定。
- 改进自定义字体下的滚动流畅度。
- 改进耗电和 CPU 占用，动画只在有变化时刷新。
- 改进设置控件一致性，各处共用同一套实现。
